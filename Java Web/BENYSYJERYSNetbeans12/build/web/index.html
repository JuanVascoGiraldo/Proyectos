La empresa BENYS Y JERYS ofrece multiples Helados que se vende por cubetas de Helados cada cubeta de helado tiene una cantidad de producto de 1 Litro de contenido, se ofrece una variedad de sabores, de los productos y se cuenta con diversas promociones que se veran reflejadas en los pedidos realizados, las promociones constan que por cierta cantidad en una compra de un producto se ofrece un procentaje de descuento del precio, los descuentos elige el usuario al momento de agregar segun sea el producto
